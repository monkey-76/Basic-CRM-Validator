# Basic CRM Validator

A Python-based Command Line Interface (CLI) tool designed to validate and categorize customer data from CSV files. 

## Features
- **Data Validation**: Uses Regular Expressions (Regex) to ensure names and 10-digit phone numbers are in the correct format.
- **Lead Categorization**: Automatically assigns a status (`lead`, `middle lead`, etc.) based on the number of customer visits.
- **CLI Support**: Supports flags for custom file input (`-c` or `--csv`).
